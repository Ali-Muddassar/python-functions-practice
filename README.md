# Python Functions Practice

## Description
A collection of Python scripts demonstrating the use and behavior of functions. This repository covers function definition, arguments, return values, and scope. Created as part of my foundational Python learning journey with **@Codanics**.

## Files and Topics Covered

| File Name | Topic | Description |
| :--- | :--- | :--- |
| `F_1.py` | Basic Function | Defining and calling a simple function. |
| `F_2.py` | Parameters | Passing arguments to a function. |
| `F_3.py` | Return Values | Returning data from a function. |
| `F_4.PY` | Scope | Understanding local and global variables. |
| `F_5.py` | Default Arguments | Using default parameter values. |
| `F_6.py` | Combined Practice | Applying multiple function concepts. |

## Purpose
To build a strong, practical understanding of Python functions—essential for writing clean, modular, and reusable code in data analysis and software development.
